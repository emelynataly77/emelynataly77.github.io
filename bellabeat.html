🔍 Bellabeat Capstone Project

Tools Used: Excel, BigQuery SQL, RStudio

📌 Objective

Analyze smart device usage data to uncover health behavior trends and provide strategic insights to Bellabeat’s marketing team.

🧹 Data Cleaning

  • Used Excel to identify and remove duplicates, inspect null values, and standardize date/time formats
  • Ran SQL queries in BigQuery to filter and summarize usage metrics
  • Exported the cleaned dataset to CSV and analyzed it in R

📊 Analysis Highlights

Using R and ggplot2, I created the following insights:

  • Afternoon hours had the highest average calories burned
  • Most users were more active Monday through Thursday

💡 Key Takeaways

  • Bellabeat should target afternoon workout enthusiasts with personalized marketing
  • Encouraging weekday engagement can boost usage frequency
